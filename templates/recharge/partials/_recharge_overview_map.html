{% comment %} Recharge overview map: every recharge site at once, on the recharge:list overview page. The Bucket-3 overview map (see docs/2.0-UX-PATTERN-SPEC.md) for the recharge screen — the recharge analogue of the Zones boundary map and the Surface Diversions point map. Recharge sites are MIXED geometry: most are polygon basins, some are single points. So this map draws both, exactly as the big district map does (templates/geography/map.html): one GeoJSON source from recharge:sites_geojson (which already carries pk in properties), with a fill+outline pair filtered to non-points and a glow+circle pair filtered to points. Clicking either opens a popup with the site name and a link to its full detail page. Modeled on _zones_overview_map.html. The parent page must provide a
with the matching map_id, and must load MapLibre GL JS + map-core.js (the map_scripts block) before this include. Variables: map_id - ID of the existing
container on the page (required) {% endcomment %}
