<task type="auto">
  <name>Task 2: Add HTMX loading indicator and toast notifications</name>
  <files>templates/base.html, static/css/app.css</files>
  <action>
1. HTMX loading indicator:
   - Add a thin progress bar at the top of the page (position: fixed, top: 0, left: 0, width: 100%, height: 3px)
   - Use HTMX events: htmx:beforeRequest shows bar, htmx:afterRequest hides bar
   - Color: --color-gold
   - CSS animation: indeterminate shimmer (left-to-right gradient)
   - Add CSS classes: .htmx-indicator (hidden by default), .htmx-request .htmx-indicator (visible)
   - Actually, use the simpler HTMX class approach: add hx-indicator to body or use htmx:configRequest

   Simplest approach using HTMX built-in:
   - Add div#loading-bar inside base.html body, before app-shell
   - CSS: .loading-bar { position:fixed;top:0;left:0;width:100%;height:3px;background:var(--color-gold);transform:scaleX(0);transform-origin:left;transition:transform 0.2s;z-index:9999 }
   - JS in base.html script block:
     document.body.addEventListener('htmx:beforeRequest', () => { document.getElementById('loading-bar').style.transform = 'scaleX(0.7)' });
     document.body.addEventListener('htmx:afterRequest', () => { const bar = document.getElementById('loading-bar'); bar.style.transform = 'scaleX(1)'; setTimeout(() => { bar.style.transition = 'none'; bar.style.transform = 'scaleX(0)'; setTimeout(() => bar.style.transition = 'transform 0.2s', 50) }, 300) });

2. Toast notification system:
   - Add div#toast-container (position:fixed;bottom:space-lg;right:space-lg;z-index:9999;display:flex;flex-direction:column;gap:space-sm)
   - Toast CSS: .toast (background:color-elevated;border:1px solid border;border-radius:radius-md;padding:space-sm space-lg;color:text-primary;font-size:0.875rem;box-shadow:shadow-pop-sm;opacity:0;transform:translateY(10px);transition:opacity 0.2s, transform 0.2s)
   - .toast.show (opacity:1;transform:translateY(0))
   - .toast-success (border-left:3px solid var(--forest-400))
   - .toast-error (border-left:3px solid oklch(0.65 0.20 25))
   - JS: window.showToast(message, type='success', duration=3000) function
   - Auto-dismiss after duration
   - Hook into HTMX events where appropriate (e.g., after successful form submissions via HX-Trigger response header)

   The Django side: views that handle create/update/delete can set HX-Trigger: {"showToast": {"message": "...", "type": "success"}} response header.
   JS listens: document.body.addEventListener('showToast', (e) => showToast(e.detail.message, e.detail.type))
  </action>
