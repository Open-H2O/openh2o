<objective>
Extract inline styles from ~40 templates into reusable CSS component classes, then add empty-state components to all list pages.

Purpose: Every template currently uses inline styles for the same patterns (detail layouts, section headers, field groups, tables, forms, badges). Output: app.css with ~15 new component classes, all templates using classes instead of inline styles, empty states on list pages.
</objective>

Add the following component class sections to app.css AFTER the existing dashboard grid section:

**Page layout classes:**
- `.page-stack` -- flex column with gap (replaces `display:flex;flex-direction:column;gap:var(--space-lg)`)
- `.page-grid-2col` -- 2-column grid (replaces `display:grid;grid-template-columns:1fr 1fr;gap:var(--space-lg)`)

**Section and field classes:**
- `.section-header` -- section title with bottom border (font-size:1rem;font-weight:600;color:primary;margin-bottom:space-lg;padding-bottom:space-sm;border-bottom)
- `.field-group` -- wrapper for label+value pairs (margin-bottom:space-md)
- `.field-label` -- uppercase label (font-size:0.75rem;font-weight:600;color:tertiary;text-transform:uppercase;letter-spacing:0.05em;margin-bottom:2px)
- `.field-value` -- value text (color:primary;font-size:0.9375rem)
- `.field-value-mono` -- monospace value (extends field-value with font-mono)
- `.field-row` -- flex row for value + edit button (display:flex;align-items:flex-start;gap:space-xs)

**Table classes:**
- `.data-table` -- full-width table with border-collapse
- `.data-table th` -- header cell (font-size:0.75rem;font-weight:600;color:tertiary;text-transform:uppercase;letter-spacing:0.05em;padding:space-sm;text-align:left;border-bottom:2px solid border)
- `.data-table td` -- body cell (padding:space-sm;border-bottom:1px solid border;font-size:0.875rem;color:text-primary)
- `.data-table tr:hover td` -- hover row (background:color-hover)
- `.data-table-link` -- clickable link in table (color:color-blue;text-decoration:none;font-weight:600)

**Form classes:**
- `.form-group` -- form field wrapper (margin-bottom:space-lg)
- `.form-label` -- label above input (display:block;font-size:0.875rem;font-weight:600;color:text-secondary;margin-bottom:space-xs)
- `.form-input` -- text input (width:100%;padding:space-sm space-md;background:color-elevated;border:1px solid border;border-radius:radius-md;color:text-primary;font-size:0.875rem;font-family:font-display)
- `.form-input:focus` -- focus state (border-color:color-gold;outline:none;box-shadow:0 0 0 2px gold-muted)
- `.form-select` -- select dropdown (same as form-input plus appearance:none with chevron background)
- `.form-textarea` -- textarea (same as form-input with min-height:120px;resize:vertical)
- `.form-help` -- help text below input (font-size:0.8125rem;color:text-tertiary;margin-top:space-xs)

**Navigation classes:**
- `.back-link` -- breadcrumb-style back link (color:text-secondary;text-decoration:none;font-size:0.875rem)
- `.back-link:hover` -- hover state (color:text-primary)

**Badge/status classes:**
- `.badge` -- base badge (display:inline-flex;align-items:center;padding:2px 8px;border-radius:radius-sm;font-size:0.75rem;font-weight:600)
- `.badge-green` -- healthy/active (background:oklch(0.45 0.12 155/0.2);color:var(--forest-400))
- `.badge-orange` -- warning (background:oklch(0.45 0.12 55/0.2);color:var(--furnace-400))
- `.badge-red` -- critical/inactive (background:oklch(0.45 0.12 25/0.2);color:oklch(0.65 0.20 25))
- `.badge-blue` -- info (background:oklch(0.45 0.12 240/0.2);color:var(--color-blue))

**Empty state class:**
- `.empty-state` -- centered empty message (text-align:center;padding:space-xl;color:text-tertiary)
- `.empty-state-icon` -- large icon above text (font-size:2rem;margin-bottom:space-md;opacity:0.5)
- `.empty-state-text` -- message (font-size:0.9375rem;margin-bottom:space-md)
- `.empty-state-action` -- CTA link below message (uses btn-primary styling)

**Inline edit classes:**
- `.edit-btn` -- pencil button for inline editing (background:none;border:none;cursor:pointer;color:text-tertiary;padding:2px 4px;border-radius:radius-sm;transition:color 0.15s)
- `.edit-btn:hover` -- hover state (color:color-gold)

**Misc:**
- `.text-secondary` -- secondary text color shorthand
- `.text-tertiary` -- tertiary text color shorthand
- `.text-mono` -- monospace font shorthand
- `.border-separator` -- top border separator (margin-top:space-lg;padding-top:space-sm;border-top:1px solid border)
- `.item-row` -- list item with bottom border (padding:space-xs 0;border-bottom:1px solid border)

Read tokens.css to verify EVERY variable name before writing. Do NOT use variable names from plan spec without confirming they exist.
