<objective>
Rename "Allocation" to "Water Budget" everywhere a user can see it, and add plain-language
explanations of how the ledger works as a balance sheet (budget vs usage).

Purpose: "Allocation" is jargon. First-time GSA admins read "Water Budget" and immediately
understand it's the amount assigned to an area for a period. This is a display-only rename
(decided 2026-05-28): every visible label changes, but internal code — URL names
(`allocations_list`), the `AllocationPlan` model class, and the `"allocation"` DB value in
`source_type` — stays put. Zero migrations, zero broken links.
Output: Relabeled templates/sidebar/forms/model choice labels, a new glossary entry, and
contextual help text clarifying the budget-vs-usage framing.
</objective>
