## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] CSS @import bypassed cache busting**
- **Found during:** Task 6 (visual checkpoint verification)
- **Issue:** tokens.css loaded via CSS @import in app.css didn't respect the ?v= cache parameter, so browser served stale tokens
- **Fix:** Moved tokens.css to a separate `<link>` tag in base.html with its own ?v= parameter
- **Committed in:** `3a84e9d`

**2. [Rule 1 - Bug] Infrastructure form inputs had wrong CSS class**
- **Found during:** Task 6 (checkpoint, user-reported screenshot)
- **Issue:** Infrastructure templates used `input-field` class (doesn't exist) instead of `form-input`
- **Fix:** Replaced all `input-field` with correct `form-input`/`form-select`/`form-textarea` classes
- **Committed in:** `7e402d7`

**3. [Rule 1 - Bug] Hero stat numbers used code font with dotted zeros**
- **Found during:** Task 6 (checkpoint, user-reported screenshot)
- **Issue:** `.stat-card-value` used JetBrains Mono (monospace) which has dotted zeros, looked wrong for display numbers
- **Fix:** Changed to Public Sans display font with `font-variant-numeric: tabular-nums` for alignment
- **Committed in:** `7e402d7`
