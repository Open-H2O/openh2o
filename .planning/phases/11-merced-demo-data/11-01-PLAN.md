If the MAINDRIVE data format is different from Tulare (different field names), adapt accordingly but preserve parcel number and area fields in the output.

  - max_rate_cfs: 10-500 range (MID PODs can be large)

  Create WaterRightParcel links connecting rights to nearby parcels.

  Create DiversionRecord entries:
  - Monthly for last 12 months
  - Seasonal pattern: higher Apr-Sep, lower Nov-Feb
  - Volumes consistent with face values (50-80% utilization)

  Create RechargeSite records for the Merced area:
  - "Merced River Recharge Basin" (spreading_basin, near Snelling)
  - "El Nido Spreading Grounds" (spreading_basin, near El Nido)
  - "Bear Creek Recharge" (streambed, along Bear Creek)
  - "Le Grand ASR Well" (asr_well, near Le Grand)

  Set locations to realistic coordinates within the subbasin. Link each to appropriate zone via zone FK.

  Create RechargeEvent records:
  - 2-4 events per site in the last 12 months
  - Volumes: 100-3,000 AF per event
  - Wet season bias (Dec-Apr)
  - water_type: "SW" for surface recharge, "ST" for storm events

  **Accounting data:**
  Use existing ReportingPeriod records (WY 2024-2025, WY 2025-2026) from Kaweah seed, or create if they don't exist (get_or_create).

  Create WaterAccount records (10):
  - account_number: "MER-ACCT-001" through "MER-ACCT-010"
  - Named after Merced-area farming operations
  - Link 3-5 parcels to each account via WaterAccountParcel

  Create AllocationPlan records:
  - One per zone per water type per reporting period
  - GW allocations: 1.5-2.5 AF/acre (Merced has tighter allocations than Kaweah due to overdraft severity)

  Create ParcelLedger entries (~1,200+):
