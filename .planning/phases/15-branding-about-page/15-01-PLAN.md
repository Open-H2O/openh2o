  1. Create static/img/ directory.
  2. Generate a professional water-themed logo via mcp__llm-gateway__llm_image. Prompt should request: minimalist dark-background logo for "OpenH2O" water accounting platform, water drop or flow motif, California Gold (#E4A317) as primary accent with Pacific Blue (#1B7FAF) secondary, clean geometric style suitable for a government/civic tech application, no text in the image (text will be added separately). <task type="auto">
  <name>Task 2: Build About page with policy timeline</name>
  <files>config/views.py, config/urls.py, templates/about.html, templates/partials/_sidebar.html, static/css/app.css</files>
  <action>
  1. Add an `about` view to config/views.py. No login required (public page). Context: pass logo_exists boolean (check if static/img/logo.png exists).
  2. Add URL route: path("about/", about, name="about") in config/urls.py. Place before the catch-all "" index route.
  3. Create templates/about.html extending base.html with these sections:

  **Hero section:**
  - OpenH2O name + tagline: "Open-source water accounting for California"
  - One-paragraph purpose statement derived from PROJECT.md core value: A poorly-funded agency buys a frontier AI subscription, points it at this repo and a $15/mo VPS, and the AI stands the platform up. Access is the product, not features.
  - If logo exists, display it (max-width 200px, centered above the tagline)

  **Policy Timeline section** (vertical timeline, CSS-only):
  Timeline entries (chronological):
  - 2014: SGMA signed — Sustainable Groundwater Management Act requires ~200 GSAs to manage 94 critically over-drafted basins
  - 2016: AB 1755 — Open and Transparent Water Data Act mandates public data integration
  - 2019: First GSPs due — Groundwater Sustainability Plans filed, exposing data infrastructure gaps in small agencies
  - 2022: GEARS modernized — DWR updates Groundwater Extraction Annual Reporting to digital CSV format
  - 2023: CalWATRS launched — Water Board opens California Water Transfer Reporting System for surface diversions
  - 2024: OpenET operational — Satellite ET data becomes production-ready for water budgets
  - 2025: Newsom EOs — Executive orders accelerate water infrastructure modernization and data sharing
  - 2026: OpenH2O — Open-source platform built so any agency can deploy with AI assistance on a $15/mo server

  **Quick Access section:**
  Two cards linking to Getting Started and Glossary (reuse dashboard-card pattern). Only show if user is authenticated (these pages require login).

  **Credits section:**
  Built by Brent Vanderburgh. Open source. Powered by Django, PostGIS, HTMX, MapLibre. Designed with the VanderDev design system.

  4. Add CSS for the timeline component in app.css:
  - Vertical line on the left (2px, gold accent)
  - Year badges (gold background, dark text, inline-block)
  - Event descriptions in card-style containers
  - Responsive: timeline works on mobile (full-width cards, left-aligned)
  - Use existing OKLCH tokens for all colors

  5. Add "About" link to the sidebar Help section in _sidebar.html, below Glossary. Use an info-circle SVG icon. The link should be visible to all users (authenticated and anonymous) since the About page is public.
  </action>
