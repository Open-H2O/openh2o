## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] HTMX indicator "Processing..." visible when idle**
- **Found during:** Task 3 (visual checkpoint)
- **Issue:** HTMX 2.x doesn't inject default `.htmx-indicator` styles, so indicator text was always visible
- **Fix:** Added .htmx-indicator opacity rules to app.css
- **Committed in:** f443556

**2. [Rule 1 - Bug] [hidden] attribute overridden by display: flex**
- **Found during:** Task 3 (visual checkpoint)
- **Issue:** `.toolbar-row { display: flex }` overrode the browser's `[hidden] { display: none }` rule
- **Fix:** Added `[hidden] { display: none !important }` global rule
- **Committed in:** abff79d

**3. [Rule 1 - Bug] Duplicate map legend**
- **Found during:** Task 3 (user reported)
- **Issue:** Both "Layers" panel and "Features" legend showed identical items
- **Fix:** Removed redundant `legend` config from MAP_CONFIG
- **Committed in:** f443556
