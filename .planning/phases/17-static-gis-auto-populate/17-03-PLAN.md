**API endpoints verified (2026-05-25):**

1. USGS 3DHP MapServer Layer 50 (Flowline):
   URL: `https://hydro.nationalmap.gov/arcgis/rest/services/3DHP_all/MapServer/50`
   Key fields: gnisidlabel (name), featuretypelabel, lengthkm, streamorder, id3dhp
   Geometry: esriGeometryPolyline (uses "paths" not "rings")
   Max record count: 2,500 per page
   Spatial ref: EPSG:3857 (request outSR=4326)
   Pagination: supported

2. Census TIGERweb Counties Layer:
   URL: `https://tigerweb.geo.census.gov/arcgis/rest/services/TIGERweb/State_County/MapServer/1`
   Key fields: NAME, GEOID, STATE, COUNTY, BASENAME
   Geometry: esriGeometryPolygon
   Max record count: 100,000 (all 58 CA counties in one page)
   Filter: STATE='06' for California

**Geometry note:** Current arcgis.py only handles polygons (rings). Flowlines are polylines (paths). Need `esri_polyline_to_geos` function.
</context>
