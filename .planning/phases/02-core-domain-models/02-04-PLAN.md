  2. Rebuild the Docker image: docker compose up -d --build
  3. Run: docker compose exec web python manage.py makemigrations core geography parcels wells measurements accounting surface recharge datasync reporting health
  4. If circular dependency errors occur:
     - Check which FKs cause the cycle
     - Break the cycle by ensuring one side uses string reference with lazy=True

  7. Verify: docker compose exec db psql -U openh2o -d openh2o -c "\dt" | wc -l (should show ~48 tables plus Django built-in tables)
