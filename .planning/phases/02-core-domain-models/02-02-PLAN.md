**Key pattern — ParcelLedger double-entry (from Rio):**
Supply entries are positive, usage entries are negative. Every water movement is a ledger entry with source type tracking. This is the single source of truth for water accounting.

**Key pattern — Dual SRID geospatial:**
Store WGS84 (EPSG:4326) for web display. Native SRID configured in SiteConfig for local accuracy.
