### Auto-fixed Issues

**1. [Rule 1 - Bug] List sort crash on mixed types without created_at**
- **Found during:** Checkpoint (user testing)
- **Issue:** PointOfDiversion has no created_at field; sort compared int pk with datetime
- **Fix:** Use epoch fallback datetime for PODs
- **Committed in:** `052c0de`

**2. [Rule 1 - Bug] Draw mode buttons not disabled on initial page load**
- **Found during:** Checkpoint (user testing)
- **Issue:** updateDrawModeForType only ran on radio change, not on page init
- **Fix:** Call on init + guard against map not yet created
- **Committed in:** `13132c3`, `0000262`

**Total deviations:** 2 auto-fixed bugs (both from checkpoint user testing), 0 deferred
**Impact on plan:** Both fixes necessary for correct UX. No scope creep.
