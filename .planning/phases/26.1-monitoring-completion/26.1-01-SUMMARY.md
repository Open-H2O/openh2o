## Decisions Made
- Rewrote DWR WDL and DWR SGMA adapters entirely. The original endpoints (wdl.water.ca.gov/waterdatalibrary and sgma.water.ca.gov/webservice/SGMA) both return 404. California Natural Resources Agency consolidated groundwater data into the CNRA Open Data Portal CKAN datastore (data.cnra.ca.gov). The DWR SGMA adapter filters by monitoring_program='SGMA' on the same Periodic Groundwater Level Measurements dataset.
- Used lazy importlib imports in registry.py to avoid circular imports with the adapter registration pattern in __init__.py.
- Skipped CIMIS adapter (ISS-007 remains open) since it requires API key registration and the platform works without it (OpenET provides ET data instead).
